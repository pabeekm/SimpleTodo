# Pre-work - SimpleTodo

SimpleTodo is an android app that allows building a todo list and basic todo items management functionality including adding new items, editing and deleting existing items. This README was copied from CodePath's README template.

Submitted by: Patricia Beekman
Time spent: 3 hours spent in total

## User Stories

The following required functionality is completed:

[X] User can successfully add and remove items from the todo list.
[X] User can tap a todo item in the list and bring up an edit screen for the todo item and then have any changes to the text reflected in the todo list.
[X] User can persist todo items and retrieve them properly on app restart.
